# WhosTheDumbass.com

## Overview
Viral IQ leaderboard where users get scored based on ChatGPT conversation history.
Score below 140? "TOO STUPID FOR LINKS"

## Features
- IQ test via encoded prompt (like DNA Chat)
- 30+ categories analyzed
- 3-minute timer
- One attempt per IP
- Giant leaderboard
- Username + links (140+ IQ only)
- Crypto payments for "STUPID" / "DUMB" badges
- AgentPixels-style UI

## Tech Stack
- Frontend: Static HTML/CSS/JS (GitHub Pages)
- Backend: Vercel serverless
- Crypto: SOL/USDC wallet
- Domain: WhosTheDumbass.com
